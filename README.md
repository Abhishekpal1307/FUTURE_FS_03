# ☕ Urban Brew Animations – Local Business Website

A modern and responsive business website developed for **Urban Brew**, a premium café located near **Lodhi Garden, New Delhi, India**. This project was created as part of **Task 3 – Local Business Website & Live Pitch Project** under the **Future Interns Full Stack Web Development Internship**.

The website is designed to enhance the café’s online presence, attract more customers, and provide a seamless browsing experience with engaging UI animations and interactive features.

---

## 🚀 Live Project Purpose

Urban Brew needed a professional website to:

* Build a strong online identity
* Showcase menu items with pricing
* Display café ambiance through gallery images
* Help customers locate the café easily
* Enable table bookings and future online order management
* Improve trust and customer engagement

---

## ✨ Key Features

### 🏠 Modern Homepage

* Attractive hero section
* Smooth scrolling navigation
* Premium café branding design

* Detailed menu with food/drink categories
* Product images for every menu item
* Dual currency pricing:

  * ₹ Indian Rupees
